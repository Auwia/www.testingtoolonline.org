Can't create more than 4 primary partitions

The partition is on the left and you reached already the number of maximum partitions allowed?

Can't create more than 4 primary partitions? This is a very common error on dynamics hard-disk where the final plan is never clear! :-)

To solve quickly the problem is necessary to unmount the primary partition (from Linux) because from Windows I couldn't find any solution or working alternatives.

To unmount the primary partition I would recommend a Linux Live DVD or USB live distribution. I personally used lubuntu https://lubuntu.net/because it is light and it contains by default GParted (but I guess all distros they have), so you can use which one you prefer.

After booted my computer I started gparten and I just moved my partition on the left and I did a resize with the unused partition that was on the left. To run GParted is only sufficient to type from the command line:

sudo gparted

As mount point I didn't selected any thing and after 40 minutes (without any interaction) my hard-disk was ready to reboot.

After the reboot everything was working and group was still displaying the right configuration.
