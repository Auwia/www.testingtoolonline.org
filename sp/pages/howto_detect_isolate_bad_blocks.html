Detectar y Corregir Sectores Defectuosos

1. Detectar Sectores Defectuosos

El comando badblocks examina los dispositivos de almacenamiento.

sudo badblocks -sv /dev/sdX
sudo badblocks -wvs /dev/sdX

2. Guardar Sectores Defectuosos

Guarde los sectores defectuosos en un archivo:

sudo badblocks -o badblocks.txt /dev/sdX

3. Crear un Sistema de Archivos

Cree un sistema de archivos excluyendo sectores defectuosos.

sudo mkfs.ext4 -l badblocks.txt /dev/sdX1
sudo mkfs.ext4 /dev/sdX1

4. Montar y Probar

sudo mount /dev/sdX1 /mnt
sudo dd if=/dev/zero of=/mnt/testfile bs=1M count=100
sudo umount /mnt

5. Monitorear Sectores Defectuosos

sudo badblocks -sv /dev/sdX

Recommendations

  • Reemplace dispositivos con muchos sectores defectuosos.
  • Realice copias de seguridad regularmente.
