Détection et Réparation des Secteurs Défectueux

1. Détection des Secteurs Défectueux

La commande badblocks examine les périphériques de stockage.

sudo badblocks -sv /dev/sdX
sudo badblocks -wvs /dev/sdX

2. Sauvegarder les Secteurs Défectueux

Enregistrez les secteurs défectueux dans un fichier:

sudo badblocks -o badblocks.txt /dev/sdX

3. Créer un Système de Fichiers

Créez un système de fichiers excluant les secteurs défectueux.

sudo mkfs.ext4 -l badblocks.txt /dev/sdX1
sudo mkfs.ext4 /dev/sdX1

4. Monter et Tester

sudo mount /dev/sdX1 /mnt
sudo dd if=/dev/zero of=/mnt/testfile bs=1M count=100
sudo umount /mnt

5. Surveiller les Secteurs Défectueux

sudo badblocks -sv /dev/sdX

Recommendations

  • Remplacez les appareils avec de nombreux secteurs défectueux.
  • Sauvegardez vos données régulièrement.
