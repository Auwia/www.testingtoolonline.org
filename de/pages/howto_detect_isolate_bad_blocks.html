Erkennen und Beheben beschädigter Sektoren

1. Beschädigte Sektoren erkennen

Der Befehl badblocks untersucht Speichergeräte.

sudo badblocks -sv /dev/sdX
sudo badblocks -wvs /dev/sdX

2. Beschädigte Sektoren speichern

Speichern Sie die beschädigten Sektoren in einer Datei:

sudo badblocks -o badblocks.txt /dev/sdX

3. Erstellen eines Dateisystems

Erstellen Sie ein Dateisystem, das beschädigte Sektoren ausschließt.

sudo mkfs.ext4 -l badblocks.txt /dev/sdX1
sudo mkfs.ext4 /dev/sdX1

4. Partition einhängen und testen

sudo mount /dev/sdX1 /mnt
sudo dd if=/dev/zero of=/mnt/testfile bs=1M count=100
sudo umount /mnt

5. Überwachen beschädigter Sektoren

sudo badblocks -sv /dev/sdX

Recommendations

  • Ersetzen Sie Geräte mit vielen beschädigten Sektoren.
  • Sichern Sie Ihre Daten regelmäßig.
